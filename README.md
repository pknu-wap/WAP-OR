## WAP-OR: 와포어
> 함께 절약해요! 가계부 커뮤니티 앱

와포어는 오픈채팅 거지방을 모티브로 한 소비 기록을 공유하는 커뮤니티 기반 가계부 앱입니다.

지출이 있을 때마다 페이 로그를 작성하여 공유하고, 댓글을 남기며 자유롭게 소통할 수 있어요.

가상 계좌를 통해 소비 목표 금액을 설정하여 계획적인 소비를 도와줘요.
