![와포어 메인](https://github.com/pknu-wap/WAP-OR/blob/main/docs/main.png?raw=true)


## 서비스 소개
와포어는 오픈채팅 거지방을 모티브로 한 소비 기록을 공유하는 커뮤니티 기반 가계부 앱입니다.



## 개발 환경
- Android Studio:
- Java: 17
- Spring Boot: 3.3.4
- Gradle: 8.10.2
- 협업 툴: Notion, Figma
